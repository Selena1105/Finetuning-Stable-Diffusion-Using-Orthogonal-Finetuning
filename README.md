# Finetuning-Stable-Diffusion-Using-Orthogonal-Finetuning

This project fine-tune the Stable Diffusion model for subject-driven generation using the
Orthogonal Finetuning via Butterfly Factorization(BOFT) on DreamBooth dataset.
